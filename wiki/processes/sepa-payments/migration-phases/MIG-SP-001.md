## Scope
Stand up the event backbone (Kafka), the OpenTelemetry observability stack and the ISO 20022 canonical message library. No customer-facing change in this phase; the new infrastructure runs in parallel with the legacy synchronous flow.

## Acceptance criteria
- Kafka cluster active-active across two EU regions with documented operational runbooks.
- OpenTelemetry collector receiving spans from at least three pilot services.
- ISO 20022 schema library published as an internal package, version 1.0.
- Architecture Review Board sign-off on the foundation.

## Risks
- Kafka operations is a new skill set — hiring + training is on the critical path.
- Migrating existing telemetry to OpenTelemetry surfaces incomplete instrumentation.
- ISO 20022 internal model needs vendor alignment that has cross-team dependencies.

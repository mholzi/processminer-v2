## Responsibility
Scores every payment in real time using a gradient-boosted model and a behavioural feature stack. Surfaces the top contributing features for audit and explainability.

## Technical detail
Models exported to ONNX and served by Python with ONNX Runtime. Feature store on Cassandra with a Redis hot cache for the last 24h of activity per customer. Champion-challenger A/B routing handled at the orchestrator level.

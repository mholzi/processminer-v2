## Responsibility
Validates every inbound ISO 20022 message: schema correctness, IBAN structure, BIC validity, scheme rule eligibility (SCT Inst amount cap, SDD mandate presence).

## Technical detail
Pre-loaded JSON schema set, Redis-cached scheme rule lookups, hot rule reload on change without a restart. 100ms p99 latency budget. Rejection events carry a structured reason code.

## The benchmark
ICC and SWIFT published the first industry API standards for bank guarantees and standby LCs in 2024, enabling real-time lifecycle management — issuance, advising, amendments, and claims — via a standardised API layer. Leading banks have completed live transactions as of early 2026.

## Relevance
Deutsche Bank's SWIFT MT/FIN-based transmission step (PS-BGIT-006) will need to evolve toward the ICC-SWIFT API standard as it becomes the industry baseline for real-time, multi-portal guarantee connectivity with corporate clients and beneficiary banks.

## Evidence
ICC/SWIFT joint announcement 2024; BNP Paribas-Surecomp pilot completed Nov 2024; Standard Chartered first live ICC-SWIFT API transaction Jan 2026.

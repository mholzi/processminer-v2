## The gap
No NLP-based sanctions screening engine is validated for deployment; the Compliance function must formally calibrate and sign off any replacement or augmentation of the current tool before go-live, and no such process has been initiated.

## Resolution
TD-5 drives engine selection and a parallel-operation calibration period in which the NLP engine's output is compared against the current tool's results. The Compliance function validates performance before the NLP engine replaces the existing tool; the current string-match tool remains in production until calibration is complete.

## Status
Open. Vendor selection and compliance calibration programme not yet initiated; existing tool remains primary.

## The gap
The bank has no ICC-SWIFT API integration capability today; SWIFT transmission relies on manual MT 760. Realising TS-6 requires API layer development, SWIFT connectivity changes and Trade Finance System upgrades that do not yet exist.

## Resolution
TD-7 drives the capability build: vendor selection, API layer development, SWIFT connectivity upgrade and TFS integration. A formal pre-transmission checklist within the current MT 760 workflow can serve as interim mitigation for CG-BGI-003 while the full API programme is under way.

## Status
Open. ICC-SWIFT API integration programme not yet initiated; interim mitigation for CG-BGI-003 not yet assessed.

## The idea
Adopt the ICC-SWIFT API standards to create a direct API channel between the Corporate Portal and the Trade Finance System — automating instruction receipt at intake, straight-through guarantee generation, and SWIFT transmission, with system-enforced pre-transmission field validation against the approved application.

## Expected benefit
Eliminates manual SWIFT MT798 handling, reduces SWIFT rejections from wrong BIC details, enforces pre-transmission validation and automates facility utilisation updates — closing the pre-transmission control gap.

## Feasibility
High complexity — requires API integration, SWIFT connectivity changes and Trade Finance System upgrades. BNP Paribas and Standard Chartered have demonstrated the model in production pilots from 2024 to 2026.

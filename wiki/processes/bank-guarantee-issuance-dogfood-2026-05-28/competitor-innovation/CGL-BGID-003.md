## The move
Standard Chartered completed the world's first live ICC-Swift API Standards digital bank guarantee via Komgo's multi-bank platform in January 2026, enabling real-time automated instruction receipt and STP issuance for corporate clients through a single standardised API connection.

## Relevance
ICC-Swift API standardisation eliminates manual re-keying and bilateral connectivity overhead at intake and issuance. As corporates adopt Komgo-style portals, banks without API connectivity will lose mandate to those that offer it — a direct risk to PS-BGID-001 and PS-BGID-006.

## The idea
Connect the Trade Finance System to the ICC-SWIFT API standard (published 2024), enabling corporate clients to submit guarantee requests from their ERP or treasury system with real-time STP processing. Standard-wording guarantees would be processed end-to-end without manual intake, credit re-entry, or MT760 composition steps.

## Expected benefit
Addresses both pain points simultaneously: API submission with real-time facility check eliminates the credit re-entry loop (PP-BGID-001) and standard-wording STP removes Legal review entirely for template guarantees (PP-BGID-002). Positions the bank alongside BNP Paribas and Standard Chartered in the early-adopter cohort for the ICC-SWIFT standard.

## Feasibility
Requires Finastra Trade Innovation Nexus migration or API enablement, plus an ICC-SWIFT API connector (available via Surecomp Rivo or Komgo). Significant integration effort with corporate client ERP systems. High complexity; best pursued as a phased programme, beginning with high-volume standard-wording corporates.

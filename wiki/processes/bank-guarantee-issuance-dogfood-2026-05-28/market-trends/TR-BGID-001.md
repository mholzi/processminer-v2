## The trend
ICC and SWIFT jointly published the first industry-wide API standards for bank guarantees and standby letters of credit in 2024. These standards enable structured real-time digital exchange between banks, corporates and platforms, replacing fragmented bilateral integrations and unstructured SWIFT MT messaging for guarantee workflows.

## Relevance
The process relies on SWIFT MT760 for delivery and manual portal intake. Adopting ICC-SWIFT API standards would enable straight-through processing from the corporate portal through to the beneficiary bank, shortening the issuance cycle and directly reducing the SLA miss rate.

## Evidence
Standard Chartered completed the first live ICC-SWIFT API digital bank guarantee via Komgo in January 2026; ICC-SWIFT API standards introduced 2024 as the first industry-wide bank guarantee API specification.

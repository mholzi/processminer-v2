## The journey
Via Komgo's Konsole API, a corporate treasury system or ERP submits a structured guarantee request machine-to-machine directly to the issuing bank; the bank returns a digital instrument certificate with real-time status data, eliminating the corporate portal login entirely for high-volume clients.

## Relevance
For corporates issuing guarantees at volume — exactly the ~200/month segment this process serves — API-embedded issuance reduces data-entry errors and turnaround time to near-zero, creating an expectation that banks will accept structured API requests rather than PDF uploads or portal forms.

## Evidence
MUFG Americas integrated Konsole API in 2025 to strengthen digital trade finance workflows; Standard Chartered executed the first live ICC-Swift API bank guarantee transaction via Konsole in January 2026.

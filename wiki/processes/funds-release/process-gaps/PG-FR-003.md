## The gap
It is not defined whether a partial release against a single request is permitted. The question bites at two points: at PS-FR-002 when the CP-FR-001 limit check finds the amount exceeds the available limit, and at PS-FR-005 when Treasury funding is only partly available.

## Impact
Without a rule, analysts have no guidance when a request could be released in part — so the whole request is failed for a limit shortfall, or deferred for partial funding, when a partial release might have served.

## Next step
Confirm with the Head of Payment Operations whether partial releases against one request are allowed, and how a limit shortfall and partial funding are each to be handled.

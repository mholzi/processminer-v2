## The gap
The overview and metric M-FR-001 describe a straight-through (STP) release path with a two-hour turnaround, but the eight-step process spine documents only the manual path — no step or branch shows where an STP-eligible clean item is validated, screened and approved automatically.

## Impact
The automated path that handles the bulk of clean releases is undocumented, so its checks, controls and exception routing cannot be reviewed, audited or improved with confidence.

## Next step
Document the STP path as an explicit branch or step set — its automated validation, screening and approval, and the conditions under which an item drops out of STP into manual handling.

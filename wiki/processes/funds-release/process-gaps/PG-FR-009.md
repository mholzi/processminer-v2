## The gap
The As-Is has three turnaround metrics — M-FR-001, M-FR-002 and M-FR-003 — each carrying a target but no measured actual. Current performance against any of these targets is not measured.

## Impact
Without measured actuals there is no quantitative baseline of how the release process performs today, so target breaches go undetected and any future improvement cannot be evidenced.

## Next step
Instrument the process to measure actual turnaround for STP releases, manual releases and exception resolution, and populate each metric's value and trend.

## The move
Napier AI's Continuum platform pairs AI-driven screening with a sandbox in which financial-crime teams test rule and threshold changes before promoting them to production, and uses machine learning to suppress repeat false-positive patterns.

## Relevance
Funds Release routes every potential hit to Compliance with no learning loop, so the same false matches recur item after item. A screening capability that tunes safely in a sandbox and learns from past decisions would cut repeat adjudication on this process.

## Evidence
Napier AI's Continuum offers a sandbox testing environment for rule changes and machine-learning features that reduce false positives and refine alert patterns.

## Description
Treasury cannot confirm sufficient funding for the requested value date on a release that requires liquidity confirmation.

## Handling
The release is deferred; Treasury and the front office are notified, and the release is rescheduled to the next available value date.

## Impact
The release does not settle on the requested value date and is pushed to the next available value date.

## The idea
Instrument the release workflow to capture actual turnaround for STP releases, manual releases and exception resolution in real time, feeding a dashboard that reports each metric's value and trend against target.

## Expected benefit
The process gains a measured baseline, target breaches are detected as they happen, an intra-day manual-release target can be set and tracked, and future improvements can be evidenced quantitatively.

## Feasibility
Moderate effort — the workflow tool already timestamps each step, so the work is extraction, aggregation and reporting; depends on defining the intra-day manual target and agreeing the metric definitions.

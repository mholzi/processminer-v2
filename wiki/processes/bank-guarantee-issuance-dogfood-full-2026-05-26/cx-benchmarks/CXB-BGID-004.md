## The benchmark
The ICC-SWIFT API standards for bank guarantees, introduced in 2024 as the first industry-wide API specifications in trade finance, enable straight-through processing and real-time interoperability between banks, corporates, and multi-bank platforms — establishing a new digital-delivery baseline.

## Relevance
A guarantee process relying on manual instruction intake, email, or fax now falls below the emerging digital standard. Leading banks have already embedded ICC-SWIFT API connectivity into live client workflows and corporate clients of those banks will expect this capability as a baseline.

## Evidence
Standard Chartered press release, January 2026 — first live guarantee transaction under ICC-SWIFT API standards via Komgo; BNP Paribas completed a comparable pilot in November 2024.

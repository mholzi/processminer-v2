## Responsibility
Performs the operational steps of the closure process from request logging through mandate verification, obligations checking, disbursement, execution and archiving.

## In this process
Responsible for logging the initial closure request in the Client Lifecycle Workflow Tool (step 1); responsible for mandate verification (step 2), obligations check (step 3), and residual balance disbursement including callback (step 5); executes the closure in the Core Banking System (step 7); sends the client confirmation and archives the file (step 8).

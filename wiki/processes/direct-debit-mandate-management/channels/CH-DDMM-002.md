## Description
Secure file transfer path used by large corporate creditors to submit high-volume bulk mandate files; files are ingested automatically by the Creditor Portal.

## Usage
Used exclusively by large corporate creditors submitting volumes too large for browser-based upload. No public API exists for mandate submission, making this the only programmatic submission route.
